Four design directions.

Wiki Loop · visual explorations · pick one to develop
01 / LIGHT / MAXIMALIST

Codex

Illuminated manuscript · academic journal

Parchment paper, sepia ink, hand-drawn graph, rubricated headings, drop caps, ornamental rules. Reads like an 18th-century encyclopedia volume — knowledge as printed matter, rabbit holes as folios.
02 / DARK / RETRO-FUTURIST

Terminal//79

1979 CRT cyberdeck · phosphor amber

Amber phosphor on black with scanlines, CRT bezel, blinking cursor, ASCII-art graph, data tables, boot-sequence command prompt. Wikipedia as if queried from a 1979 mainframe. Fun, nerdy, unforgettable.
03 / DARK / REFINED EDITORIAL

Salon

Deco magazine · deep forest & gold

Luxurious editorial magazine. Deep forest green, cream type, gold hairline rules, DM Serif Display headlines, constellation-chart graph. Reads like an art journal for the intellectually curious.
04 / LIGHT / BRUTALIST

BRUTE

Swiss poster · raw grid · alert red

Brutalist Swiss poster. 170px Anton headlines, visible grid, black squares for nodes, rotated labels, stamps and overprints, data tables with yellow hover. Aggressive, loud, unapologetic.
